# ROGUE BALL

A neon arcade tube-runner built in **Unity 6 (URP)** for WebGL.

You are a rogue ball speeding through an endless neon tunnel, dodging Pong paddles that want you back in the game. Survive 5 levels of rising speed and shrinking gaps, and escape through the portal at the end.

**The twist:** the ball never actually moves forward — the world moves toward it. The scrolling grid, advancing paddles, and camera lag sell the illusion of speed.

## How to play

| Input | Action |
|-------|--------|
| **A / D** or **← / →** | Roll around the tube wall (full 360°) |

- You have **3 hearts**. A paddle hit costs 1 heart (with brief invincibility after).
- **Heart pickup** (red orb): +1 life, max 3.
- **Shield pickup** (gold cube): 3 seconds of immunity.
- Survive **5 levels × 30 seconds**, then reach the **escape portal** to win.

## Tech notes

- **Everything is procedural / self-made** — no external assets, zero license risk:
  - Tube, escape-portal torus, and all meshes generated in code
  - Grid texture, ball glow, and UI sprites created in-project
  - Every sound effect and the music track generated by code and saved as WAV
- Math-based ball controller (no physics) — position computed from an angle around the tube, fully predictable and tunable
- Unity 6, Universal Render Pipeline: HDR emission + bloom for the neon look, depth-of-field blur behind menus
- New Input System; TextMeshPro UI
- WebGL build with gzip compression, published on itch.io
